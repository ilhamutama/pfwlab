# pfwlab

pfwlab is a project to document the learning process of PHP frameworks.

## Frameworks

- Symfony http://symfony.com/
- Laravel http://laravel.com/
- Phalcon http://phalconphp.com/

## Goals

- A hello world
- Basic crud with sqlite
- Authentication
- Internationalization
- Bootstrap
